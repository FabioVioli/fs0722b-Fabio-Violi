La proprietà Z-Index

Più alto è il valore di Z-Index che do, più starà sopra
Più basso è il valore di Z-Index che do, più starà sotto
A parità di Z-Index, segue la regola del cascade - quindi green prevarrà su blue
Con Z-Index -1, l'immagine ( fuori dal div, e con Position: Absolute ) finisce sotto a tutto. anche il testo ( vedi h2 )


Valori possibili
Auto - Valore uguale a tutti i parents
Numero - qualsiasi numero intero positivo o negativo
Initial - valore di default (0)
Inherit - Ereditato dal genitore
